# DnD Fight Tracker

A lightweight, browser-based initiative tracker for Dungeon Masters.  
Track initiative order, HP, AC, and status effects for all combatants — no installation required.

## Usage

1. Open `main.html` in your browser or open the [Github Pages](https://fritizee.github.io/dnd_fight_tracker/main.html) version
2. Add combatants with their stats
3. Press the button and let the battle begin
